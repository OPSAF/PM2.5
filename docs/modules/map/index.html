📊 直角坐标系建模方法说明

原点设定
保定市徐水区 (115.5°E, 38.8°N)
X轴定义
东西方向 (东正西负)
Y轴定义
南北方向 (北正南负)
坐标单位
千米 (km)
插值方法
克里金插值 (Kriging)
可视化引擎
Apache ECharts

📐 坐标转换公式

x = R × (λ - λ₀) × cos(φ₀)
y = R × (φ - φ₀)
其中 R=6371km (地球半径), λ为经度, φ为纬度, (λ₀,φ₀)为原点

📈 回归模型设定

z_it = α₀ + β₁xᵢ + β₂yᵢ + β₃xᵢyᵢ + μᵢ + λₜ + εᵢₜ
其中 z 为 PM2.5 浓度, x,y 为直角坐标, μᵢ 为城市固定效应, λₜ 为时间固定效应
