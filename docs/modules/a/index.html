异常值处理方法

3σ原则缩尾处理（Winsorization）

对于超出 [μ-3σ, μ+3σ] 范围的极端值，进行缩尾处理：
下限 = μ - 3σ → 上限 = μ + 3σ
σ = 1% 水平（对应3σ覆盖约99.7%数据）

处理步骤

1. 按年份分组计算各变量的均值和标准差
2. 确定每年度的上下限
3. 将超出范围的数值替换为边界值

标准化处理方法

Z-score 标准化

Z = (X - μ) / σ

转换后变量均值为0，标准差为1，消除量纲差异，保证SHAP值横向可比

Pearson & Spearman 相关系数

Pearson相关系数：衡量线性相关程度
Spearman相关系数：衡量单调相关程度，对异常值更鲁棒
|r| < 0.3：弱相关 0.3 ≤ |r| < 0.7：中等相关 |r| ≥ 0.7：强相关

关键发现

  • 煤炭vs天然气：r = -0.986，强负相关（能源结构此消彼长）
  • 煤炭vs一次电力：r = -0.984，强负相关
  • PM2.5vs煤炭：r = 0.653，中等正相关（煤炭占比越高，污染越重）
  • PM2.5vs天然气：r = -0.668，中等负相关（天然气比例越高，污染越轻）
  • 风速分量u10 vs v10：r = -0.652，中等负相关（风向特性）
