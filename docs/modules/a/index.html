异常值处理方法

3σ原则缩尾处理（Winsorization）

对于超出 [μ-3σ, μ+3σ] 范围的极端值，进行缩尾处理：
下限 = μ - 3σ → 上限 = μ + 3σ
σ = 1% 水平（对应3σ覆盖约99.7%数据）

处理步骤

1. 按年份分组计算各变量的均值和标准差
2. 确定每年度的上下限
3. 将超出范围的数值替换为边界值

标准化处理方法

Z-score 标准化

Z = (X - μ) / σ

转换后变量均值为0，标准差为1，消除量纲差异，保证SHAP值横向可比

平稳性检验方法

LLC检验（Levin-Lin-Chu）

H0: 存在单位根（非平稳）→ p值 < 0.05 则拒绝H0，认为平稳

IPS检验（Im-Pesaran-Skin）

H0: 存在单位根（非平稳）→ p值 < 0.05 则拒绝H0，认为平稳

多重共线性检验

VIF检验（方差膨胀因子）

VIF = 1 / (1 - R²)
VIF < 10：无严重共线性
VIF ≥ 10：存在严重共线性，需剔除
