核心发现

基于2014-2024年京津冀地区13个城市的数据分析，我们发现以下关键结论：

显著驱动因素

环境关注度、煤炭占比、工业氮氧化物排放、第二产业占比等因素对PM2.5有显著影响。

空间溢出效应

相邻地区的污染排放对本地PM2.5浓度存在显著影响，验证了空间杜宾模型的适用性。

清洁能源效应

天然气占比和电力及其他能源占比有助于降低PM2.5浓度，优化能源结构是关键。
